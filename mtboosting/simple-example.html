Simple example

Recovering known signals.

This simulation creates a binary outcome where the true variables are known. The outcome depends on x1 and x3, and the remaining variables are noise.

Setup

The goal is simply to check that the package interface returns the true signal variables in a logit example.

Candidate variablesx1 to x20
True signalsx1 and x3
ModelBinary logit
Simulation code 
library(MultipleTestingBoosting)

set.seed(123)

n_obs <- 250
n_candidates <- 20

X <- matrix(rnorm(n_obs * n_candidates), nrow = n_obs, ncol = n_candidates)
colnames(X) <- paste0("x", seq_len(n_candidates))

eta <- 1.5 * X[, "x1"] - 1.0 * X[, "x3"]
prob <- plogis(eta)
y <- rbinom(n_obs, size = 1, prob = prob)

delta <- 1

selected <- boosting_glm(
  y = y,
  X = X,
  link = "logit",
  pval = 0.05,
  delta = delta
)

colnames(X)[selected]
#> [1] "x1" "x3"
