Empirical example

Using BMT on transformed FRED-MD/PPM data.

This page shows the package workflow on the transformed macroeconomic package data object FREDMDPPM_transformed. The macroeconomic variables are based on the FRED-MD database of McCracken and Ng (2016), maintained through the Federal Reserve Bank of St. Louis.

Data.

The example uses a transformed monthly U.S. macroeconomic package data object and turns it into the two objects needed by the package: a response vector y and a candidate regressor matrix x.

Package dataFREDMDPPM_transformed, loaded with data("FREDMDPPM_transformed", package = "MultipleTestingBoosting").
Data typeTransformed monthly U.S. macroeconomic time-series data.
ResponsesPCEPI for the linear model; PPM for the logit model, where PPM = 1[smoothed annual inflation > 2.5%].
CandidatesAll transformed macroeconomic variables except date and PCEPI. Thus PCEPI is not a covariate.
Candidate countThe examples use 101 candidate variables after excluding date and the response PCEPI.
Function callsboosting_glm(..., link = "linear") or boosting_glm(..., link = "logit").
Logit link\(\Lambda(z)=\dfrac{1}{1+\mathrm{e}^{-z}}\).

Data source and citation.

Paper citation: McCracken, Michael W., and Serena Ng. 2016. FRED-MD: A Monthly Database for Macroeconomic Research. Journal of Business & Economic Statistics 34(4): 574–589. DOI: 10.1080/07350015.2015.1086655.

Data source: Federal Reserve Bank of St. Louis, FRED-MD and FRED-QD databases, St. Louis Fed FRED databases page. Accessed July 13, 2026.

The empirical examples use the transformed data included with the package. For the logit example, monthly PCEPI log changes are summed over 12 months, converted to percent, averaged over a trailing 12-month window, and classified as high inflation when the result exceeds 2.5%.

1. Load the package data.

Load the package and the empirical data object.

Load package data 
library(MultipleTestingBoosting)

data("FREDMDPPM_transformed", package = "MultipleTestingBoosting")
data <- FREDMDPPM_transformed

data$date <- as.character(data$date)

2. Choose an empirical model.

Choose a model and a value of delta.

Researcher-selected delta:

Delta setting 
delta <- 1
if (delta < 1) stop("delta must be greater than or equal to 1.")
What changes when delta changes?
\[c_p(n,\delta)=\Phi^{-1}\left(1-\frac{p}{2n^\delta}\right)\]

A larger delta raises the screening cutoff, so the selected model usually becomes more sparse.

Inflation linear regression.

BMT first selects the regressors, then the selected linear model is reported as:

Linear inflation model

\[\pi_t = c + \sum_{j \in \mathcal{S}_{\pi}} \beta_j x_{j,t} + u_t\]

Here \(\pi_t\) is inflation, measured by PCEPI. BMT chooses the selected set \(\mathcal{S}_{\pi}\) from the candidate variables \(x\).

Linear BMT selection 
target_name <- "PCEPI"
y <- data[[target_name]]
candidate_names <- setdiff(names(data), c("date", target_name))
x_raw <- as.matrix(data[candidate_names])
storage.mode(x_raw) <- "numeric"
keep <- is.finite(y) & complete.cases(x_raw)
y <- y[keep]
x <- scale(x_raw[keep, , drop = FALSE])

delta <- 1  # choose one of: 1, 2, 3
if (delta < 1) stop("delta must be greater than or equal to 1.")

selected <- boosting_glm(
  y = y,
  X = x,
  link = "linear",
  pval = 0.05,
  delta = delta
)

selected <- as.logical(selected)
selected_variables <- colnames(x)[selected]
selected_variables

PPM logit regression.

The PPM example converts inflation into a binary response and then applies BMT with a logit link.

Binary response

\[PPM_t = \begin{cases} 1, & \bar{\pi}_t^{(12)} > 2.5 \\ 0, & \bar{\pi}_t^{(12)} \le 2.5 \end{cases}\]

Here PCEPI is the monthly log-change series. It is converted to annual inflation and then smoothed with a trailing 12-month average, \(\bar{\pi}_t^{(12)}\). The binary response equals one only when this smoothed rate exceeds 2.5%.

Selected final logit model

\[\Pr(PPM_t = 1 \mid x_t) = \Lambda(\eta_t)\]
\[\eta_t = c + \beta_1 x_{1,t} + \cdots + \beta_k x_{k,t}\]

\(\Lambda(z)=\dfrac{1}{1+\mathrm{e}^{-z}}\). The selected variables enter the logit index after BMT screening.

The BMT step is the same workflow, but with link = "logit".

Logit BMT selection 
target_name <- "PCEPI"
candidate_names <- setdiff(names(data), c("date", target_name))
x_raw <- as.matrix(data[candidate_names])
storage.mode(x_raw) <- "numeric"
monthly_pcepi_change <- as.numeric(data[[target_name]])
annual_inflation <- 100 * as.numeric(stats::filter(monthly_pcepi_change, rep(1, 12), sides = 1))
smoothed_inflation <- as.numeric(stats::filter(annual_inflation, rep(1 / 12, 12), sides = 1))
ppm <- as.integer(smoothed_inflation > 2.5)
keep <- is.finite(ppm) & complete.cases(x_raw)
ppm <- ppm[keep]
x <- scale(x_raw[keep, , drop = FALSE])

delta <- 1  # choose one of: 1, 2, 3
if (delta < 1) stop("delta must be greater than or equal to 1.")

selected_ppm <- boosting_glm(
  y = ppm,
  X = x,
  link = "logit",
  pval = 0.05,
  delta = delta
)

selected_ppm <- as.logical(selected_ppm)
ppm_variables <- colnames(x)[selected_ppm]
ppm_variables

How to read the output.

The key output is selected_variables, which lists only the variables selected by BMT.

selectedThe raw TRUE/FALSE output from boosting_glm(); it is used to index the columns of x.
selected_variablesThe names of the selected macroeconomic variables.
deltaThe researcher-chosen threshold exponent. The examples use delta <- 1 and pass it through the common delta argument, which applies it at both stages.
summary(inflation_model)A final linear inflation regression using only the variables selected by BMT.
summary(ppm_logit)A final PPM logit model using the logit-selected variables.
Interpretation: In the inflation example, PCEPI is the response variable and the selected columns are used in the final lm(). In the PPM example, ppm is binary and the selected columns are used in the final logit glm().
